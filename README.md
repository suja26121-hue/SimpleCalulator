

🧮 Simple C++ Calculator

A simple console-based calculator built using C++. It performs basic arithmetic operations and demonstrates fundamental programming concepts like functions, loops, and conditional statements.

📌 Features
Addition, Subtraction, Multiplication, Division
Menu-driven interface
Handles invalid operator input
Prevents division by zero errors
Supports repeated calculations until user exits
🛠️ Technologies Used
C++
iostream library
🚀 How to Run
Clone the repository:
git clone https://github.com/your-username/calculator.git
Navigate to the project folder:
cd calculator
Compile the program:
g++ main.cpp -o calculator
Run the program:
./calculator
📷 Sample Output
Welcome to My C++ Calculator
Enter first number: 10
Enter operator: +
Enter second number: 5
Result: 10 + 5 = 15
