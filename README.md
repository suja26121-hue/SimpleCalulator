# 🧮 Simple C++ Calculator

A simple console-based calculator built using C++.  
It performs basic arithmetic operations and helps understand core programming concepts like functions, loops, and conditional statements.

---

## ✨ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division  
- 📋 Menu-driven interface  
- ❌ Handles invalid operator input  
- ⚠️ Prevents division by zero errors  
- 🔁 Supports multiple calculations until exit  

---

## 🛠️ Tech Stack
- C++
- iostream library

---

## 🚀 How to Run

```bash
# Clone the repository
git clone https://github.com/your-username/calculator.git

# Move into project folder
cd calculator

# Compile the program
g++ main.cpp -o calculator

# Run the program
./calculator
calculator
